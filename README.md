# Supply-Chain-Dashboard

This project showcases an **interactive Power BI dashboard** built using CSV sales datasets to analyze business performance, customer behavior, shipping trends, and revenue insights. The goal is to transform raw sales data into meaningful visual analytics that support strategic decision-making and operational improvements.

The dashboard provides a comprehensive view of sales metrics, customer segmentation, order trends, and shipping performance through intuitive visualizations and KPIs.
